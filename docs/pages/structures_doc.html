Structures

Structures are objects created to represent that type! Many of them can have special functionalities such as arithmetic operations or formatting when transformed into strings!

Color Structures

Objects used to represent colors, each parameter represents its respective channel!

  • CMYKColor:
    Used to represent colors in the CMYK model (Cyan, Magenta, Yellow, Black).
  • HexColor:
    Used to represent colors in hexadecimal format.
  • HexAColor:
    Used to represent colors in hexadecimal format with alpha channel.
  • HSBColor:
    Used to represent colors in the HSB model (Hue, Saturation, Brightness).
  • RGBColor:
    Used to represent colors in the RGB model (Red, Green, Blue).
  • RGBAColor:
    Used to represent colors in the RGB model with alpha channel (Red, Green, Blue, Alpha).
  • RYBColor:
    Used to represent colors in the RYB model (Red, Yellow, Blue). 
                    
 # -- Usage! --

 cmyk_clr = CMYKColor(
    c_channel=99,
    m_channel=0,
    y_channel=43,
    k_channel=1
 )

 hex_clr = HexColor(
    value='#03FC90'
 )

 hexa_clr = HexAColor(
    value='#03FC90FF'
 )

 hsb_clr = HSBColor(
    hue=154,
    saturation=99,
    brightness=99
 )

 rgb_clr = RGBColor(
    r_channel=3,
    g_channel=252,
    b_channel=144
 )

 rgba_clr = RGBAColor(
    r_channel=3, 
    g_channel=252, 
    b_channel=144,
    a_channel=255
 )

 ryb_clr = RYBColor(
    r_channel=3,
    y_channel=252,
    b_channel=144
 )

Geometry Structures

Objects used to represent and facilitate the understanding and calculations of sizes, coordinates, and zones!

  • Coordinates2D:
    Used to represent two-dimensional coordinates (x_coord, y_coord).
  • Coordinates3D:
    Used to represent three-dimensional coordinates (x_coord, y_coord, z_coord).
  • Size2D:
    Used to represent two-dimensional size (width, height).
  • Size3D:
    Used to represent three-dimensional size (width, height, length).
  • Zone2D:
    Used to represent two-dimensional zone (x_coord, y_coord, width, height).
  • Zone3D:
    Used to represent three-dimensional zone (x_coord, y_coord, z_coord, width, height, length). 
                    
 # -- Usage! --

 coord_2d = Coordinates2D(
    x_coord=10,
    y_coord=20.22
 )

 coord_3d = Coordinates3D(
    x_coord=10,
    y_coord=20.22,
    z_coord=30.5
 )
 
 size_2d = Size2D(
    width=10,
    height=20.22
 )

 size_3d = Size3D(
    width=10,
    height=20.22,
    length=30.5
 )

 zone_2d = Zone2D(
    x_coord=10,
    y_coord=20.22,
    width=30.5,
    height=40
 )

 zone_3d = Zone3D(
    x_coord=10,
    y_coord=20.22,
    z_coord=30.5,
    width=40,
    height=40.44,
    length=60.5
 )

Explore More

All structures can inherit some characteristics, gaining special abilities such as arithmetic operations, comparison, and many others!

How does each ability work?

Each ability inherited by a structure works slightly differently, but they all help to facilitate understanding!

  • Arithmetic:
    In arithmetic abilities, you can easily perform calculations using the structure + operator + value!

    Example: 

                                
 size = Size2D(10, 20.3)
 size = size / 2
 print(size.width, size.height)

 #>>> 5, 10.15
  • Comparative:
    Used to represent colors in hexadecimal format.
  • Formatting:
    Used to represent colors in hexadecimal format with alpha channel.
  • Indexing:
    Used to represent colors in the HSB model (Hue, Saturation, Brightness).
  • Iteration:
    Used to represent colors in the RGB model (Red, Green, Blue).
