Structures

Structures are objects created to represent that type! Many of them can have special functionalities such as arithmetic operations or formatting when transformed into strings!

Color Structures

Objects used to represent colors, each parameter represents its respective channel!

CMYK Color

Used to represent colors in the CMYK model! 

                         
 # -- Example --
 cmyk_clr = CMYKColor(
    c_channel=0,
    m_channel=0,
    y_channel=0,
    k_channel=0
 )

Hex Color

Used to represent colors in hexadecimal format! 

                     
 # -- Example --
 hex_clr = HexColor(
    value=#000000
 )

HexA Color

Used to represent colors in hexadecimal format with alpha channel! 

                     
 # -- Example --
 hexa_clr = HexAColor(
    value=#00000000
 )

HSB Color

Used to represent colors in the HSB model! 

                     
 # -- Example --
 hsb_clr = HSBColor(
    hue=360,
    saturation=100,
    brightness=100
 )

RGB Color

Used to represent colors in the RGB model! 

                     
 # -- Example --
 rgb_clr = RGBColor(
    r_channel=255,
    g_channel=100,
    b_channel=100
 )

RGBA Color

Used to represent colors in the RGB model with alpha channel! 

                     
 # -- Example --
 rgba_clr = RGBAColor(
    r_channel=255,
    g_channel=100,
    b_channel=100,
    a_channel=100
 )

RYB Color

Used to represent colors in the RYB model! 

                     
 # -- Example --
 ryb_clr = RYBColor(
    r_channel=255,
    y_channel=100,
    b_channel=100
 )
Channel Range Values
Color Values
CMYK 0 to 100 - 0 to 100 - 0 to 100 - 0 to 100
Hex 00 to FF - 00 to FF - 00 to FF
HexA 00 to FF - 00 to FF - 00 to FF - 00 to FF
HSB 0 to 360 - 0 to 100 - 0 to 100
RGB 0 to 255 - 0 to 255 - 0 to 255
RGBA 0 to 255 - 0 to 255 - 0 to 255 - 0 to 255
RYB 0 to 255 - 0 to 255 - 0 to 255

Geometry Structures

Objects used to represent and facilitate the understanding and calculations of sizes, coordinates, and zones!

Coordinates2D

Used to represent X and Y coordinates! 

                         
 # -- Example --
 coordinates2D = Coordinates2D(
    x_coord=0,
    y_coord=0
 )
